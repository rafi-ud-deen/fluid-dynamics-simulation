# Fluid Dynamics Simulation

Interactive WebGL-based fluid dynamics simulation built with React, TypeScript, and advanced shader programming.

## Features

- **Realistic Physics**: Navier-Stokes fluid simulation using Jos Stam's "Stable Fluids" method
- **Visual Effects**: Bloom effects, sunrays, and vorticity confinement
- **Interactive**: Mouse/touch controls for creating fluid splashes
- **Performance**: Optimized WebGL shaders for smooth real-time simulation

## Technologies

- React 19
- TypeScript
- WebGL/GLSL
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```
